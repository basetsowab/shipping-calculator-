# Shipping Cost Calculator

A C++ application that calculates shipping costs based on package weight and shipping rules. This project was built to practice object-oriented programming, conditional logic, modular programming, and algorithmic problem solving.

## Features

- Calculates shipping costs based on package weight
- Handles user input and validation
- Uses modular functions for cleaner code organization
- Console-based interactive program
- Demonstrates core C++ programming concepts

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Functions and Conditional Logic
- Standard Template Library (STL)

## How It Works

The program:
1. Accepts package information from the user
2. Processes shipping rules based on weight
3. Calculates the final shipping price
4. Displays the result in the console

## Example Output

```bash
Enter package weight: 15

Shipping Cost: $22.50
```

## Installation

Clone the repository:

```bash
git clone https://github.com/basetsowab/shipping-calculator-.git
```

Navigate into the project folder:

```bash
cd shipping-calculator-
```

Compile the program:

```bash
g++ main.cpp -o shipping
```

Run the executable:

```bash
./shipping
```
